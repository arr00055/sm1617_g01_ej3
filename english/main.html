Authentication.

HosteRed provides us with a simple and effective authentication service that will keep our business secure. The entire client of this service must be authenticated before making use of the service, for it will need a field of username and a password that is later stored in the database of our server when the user registers providing a mail that It will be used to send you notifications.

Each time the user authenticates and connects to our remote server, the data of his business that were stored on the last occasion will be downloaded, these will replace the previous ones allowing the user to have in each moment the last version of the data that registered On the remote server.
