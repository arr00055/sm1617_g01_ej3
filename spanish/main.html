Sistema de autenticación.

HosteRed nos proporciona un servicio sencillo y eficaz de autenticación que mantendrá nuestro negocio seguro. Todo cliente de este servicio deberá autenticarse antes de poder hacer uso del servicio, para ello necesitará introducir un campo de nombre de usuario y una contraseña que posteriormente se guardará en la base de datos de nuestro servidor remoto cuando el usuario se registre proporcionando un correo que se empleará para mandarle notificaciones.

Cada vez que el usuario se autentique y se conecte a nuestro servidor remoto, se descargarán los datos de su negocio que fueron almacenados en la última ocasión, estos sustituirán a los anteriores permitiendo al usuario disponer en cada momento la última versión de los datos que registró en el servidor remoto.

Introducción de productos.

Una vez autenticados, podemos gestionar nuestra aplicación, insertando productos en el stock cuando sea conveniente, de manera fácil e intuitiva. Podemos rellenar para cada producto los siguientes campos: [Nombre, Valor actual, Valor mínimo, Nombre del Proveedor]. El usuario podrá añadir, eliminar o modificar según necesite.

¿Qué nos ofrece en general HosteRed?.

El usuario de este servicio podrá contar con una gran información administrativa sobre su negocio en su teléfono móvil, y mientras tenga acceso a Internet podrá actualizar los diferentes datos relativos a su negocio.

Un servicio de expiración de sesión, por si la sesión se queda abierta por descuido se cerrará al cabo de un tiempo por seguridad.

La posibilidad de rotar la pantalla y que se adapten todos los elementos, para mayor comodidad.

No requiere del uso de servicios auxiliares, simplemente un correo electrónico para registrarse, y así poder notificar información por el mismo.

Una interfaz simple, rápida y apta para no sobrecargar excesivamente los recursos del teléfono móvil.
